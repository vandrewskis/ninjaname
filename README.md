# Ninjaname
Convert your name to ninja name with javascript.

# Example Usage
Find example below for converting John Doe to it's ninja name.
Use ninjaname.js from the src directory.

Html:
```

<div id="ninja"></div>
<script>
	$('#ninja').html(ninjaname("John Doe"));
</script>
```

Output:
```
Zizutochi Zuzuke
```
